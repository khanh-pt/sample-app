# Adonojs - sample app

## Back-end

### Technologies

- Postgresql >= 15
- Nodejs >= 20

### Guideline for Configuration

#### Environment variable

- Thêm các biến môi trường vào file ".env" dựa theo các key trong file ".env.example"
- Run command: `node ace generate:key`
- Start with docker: `docker-compose up`
